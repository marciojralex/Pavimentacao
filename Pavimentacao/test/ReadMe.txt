EXECUTAR

Para executar esta aplicação basta.

1. Acesse a IDE, faça a importação do projeto.
2. Botão direito no projeto.
3. Clicar em executar.

obs: Esta aplicação foi desenvolvida no NetBeans, para importar para eclipse basta seguir os seguintes passos.
obs2: Por se tratar de uma aplicação Web é necessario ter algum servidor de aplicação instalado ou configurado na IDE e na máquina.

1. Abra a IDE eclipse
2. Clique em File(arquivo)
3. Clique em import(importar)
4. Dentro da pasta "Web" busque pela opção "WAR file"
5. Clicar em Browser, procure pela pasta dist do projeto (Pavimentacao) e selecionar o arquivo .war
7. Clicar em Finish
8. Botão direito no projeto.
9. Clicar em "Run as" e depois "Run on server".


DEPLOY

Para fazer o deploy da aplicação basta ter um servidor de aplicação TomCat ou glassfish e adicionar o arquivo .war,
o arquivo .war estará na pasta \dist do projeto
